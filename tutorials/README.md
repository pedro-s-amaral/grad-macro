# Tutorials

Short ECON 503 tutorials for math refreshers, optimization, data work, and macroeconomic applications.

## Math Refresher

Files in `math-refresher/` cover derivatives and optimization:

- `univariate_derivatives.Rmd`
- `multivariate_derivatives.Rmd`
- `unconstrained_optimization.Rmd`
- `constrained_optimization.Rmd`

## Data

Files in `data/` cover downloading, filtering, and working with macroeconomic data:

- `bea_data_tutorial.Rmd`
- `fred_data_tutorial.Rmd`
- `hp_tutorial.Rmd`
- `pwt_data_tutorial.Rmd`

## Macro Topics

Files in `macro-topics/` cover applied macroeconomic models and facts:

- `dispersion.Rmd`
- `kaldor_facts.Rmd`
- `solow_model.Rmd`

The shared stylesheet for these R Markdown tutorials is `style.css`.
